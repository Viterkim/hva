# Code style

- no inline comments — code should be readable without them
- no narration comments (`// Now we...`, `// This function...`, `// Here we...`)
- no placeholder sections (`// TODO`, `// implement later`, `// add logic here`)
- no `print`/`console.log` left in for debugging
- no dead code, commented-out blocks, or unused imports
